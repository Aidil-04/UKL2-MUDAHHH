# Program Ganjil Genap
Program ini digunakan untuk menentukan apakah sebuah bilangan yang dimasukkan oleh pengguna adalah ganjil atau genap.
CARA KERJA:
Program menggunakan Scanner untuk membaca input dari keyboard.
Bilangan yang dimasukkan disimpan dalam variabel bilangan.
Program menggunakan operator modulus (%) untuk mengecek sisa pembagian bilangan dengan 2.
Jika bilangan % 2 == 0, maka bilangan adalah genap.
Jika tidak, maka bilangan adalah ganjil.
Hasilnya ditampilkan ke layar.
